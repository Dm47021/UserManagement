User Management

User Management is a GUI front end to use the limited multi-user support introduced in Android 4.1 Jellybean

It allows you to create new users, view existing users and delete users easily. New usernames can have spaces. Your device MUST be rooted for this app to function properly

You can reach the developer at raghavsood@appaholics.in

For easier function, make sure to check the "Remember" option when granting root permissions.
